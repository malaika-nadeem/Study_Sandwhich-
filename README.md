# 🥪 Study Sandwich

Study Sandwich is an AI-powered study assistant that converts your notes into interactive quizzes.

Upload your study material, choose the range, and generate MCQs and short questions instantly — making learning more active and engaging.

---

## 🚀 Features

- 📂 Upload notes in multiple formats:
  - PDF
  - DOCX
  - PPTX
  - TXT

- ✂️ Select specific page ranges (for PDFs)

- 🧠 AI-powered quiz generation:
  - Multiple Choice Questions (MCQs)
  - Short Questions with answers

- ✅ Automatic answer evaluation:
  - MCQs checked instantly
  - Short answers evaluated using AI

- 📊 Final score display

---

## 🛠️ Tech Stack

- Python  
- Streamlit (Frontend + App Framework)  
- pdfplumber (PDF processing)  
- python-docx (DOCX reading)  
- python-pptx (PPTX reading)  
- Groq API (LLM for quiz generation & evaluation)

---

## 🧠 How It Works

1. User uploads notes  
2. App extracts text from file  
3. User selects:
   - Page range  
   - Number of questions  
4. AI generates quiz in JSON format  
5. Questions are displayed interactively  
6. User answers → app evaluates → score is shown

## ▶️ How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/your-username/study-sandwich.git
````

2. Go to project folder:

```bash
cd study-sandwich
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Add your API key in `.env` file:

```
GROQ_API_KEY=your_api_key_here
```

5. Run the app:

```bash
streamlit run app.py
```
